# nix-cygwin
Building and installing the **Nix Package Manager** in **Cygwin**.

## 1
Install **Cygwin** with **wget**.

## 2
In **Cygwin console** execute:
```
wget https://raw.githubusercontent.com/MDeltaX/nix-cygwin/master/nix-install
chmod +x nix-install
./nix-install
```
